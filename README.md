## :bug: 기능 설명

- Start : 시작화면 | 사이트 로고와 간단한 설명
- Main  : 메인화면 -> 메인에서 공지글, 일반 게시글, 복권계산기 등을 볼 수 있음
- Info  : 공지사항 게시판
- Starter  : 스타터 가이드 게시판
- JobGuide : 직업별 가이드 게시판
- Map : 풍맥 등 각종 지도 확인 사이트
- DailyLotto : 일일복권 계산기
- Free : 자유게시판
- Board : 자유게시판 글 작성란
- Login : 로그인 모달 창
- Sign : 회원가입 모달 창
